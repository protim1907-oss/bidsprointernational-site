# BidsPro International Website

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

1. Push this folder to GitHub
2. Log into Vercel
3. Click Add New Project
4. Import the GitHub repository
5. Click Deploy

## Connect your domain

In Vercel:
- add bidsprointernational.com
- add www.bidsprointernational.com

Then update the DNS records in your domain registrar.
